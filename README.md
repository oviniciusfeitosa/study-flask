# Study Flask

Study case of Flask Python micro framework.

## Steps

- install **virtualenv**

```sh
sudo pip install virtualenv
```

- Install Flask in virtual environment:

```sh
virtualenv flask
flask/bin/pip install flask
```

- run application

```sh
./app.py
```
